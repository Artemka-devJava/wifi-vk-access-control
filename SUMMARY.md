## 🚀 REST API Эндпоинты

### Аутентификация (AuthController)
```
GET  /api/auth/check-access         - Проверить доступ пользователя
POST /api/auth/link-device          - Связать MAC с VK ID
GET  /api/auth/user                 - Информация об устройстве
GET  /api/auth/user-by-vk           - Информация по VK ID
```

### Статистика (StatsController)
```
GET  /api/stats/summary             - Общая статистика
GET  /api/stats/devices             - Список устройств
GET  /api/stats/access-logs         - Логи доступа
GET  /api/stats/user                - Статистика пользователя
```

## 🔧 Как запустить

## 📊 База данных

**SQLite** с **4 таблицами:**

1. **users** - Информация об устройствах
   - MAC адрес (уникальный)
   - VK User ID
   - Статус подписки
   - Время подключения

2. **access_logs** - Логи доступа
   - Время доступа
   - IP адрес, URL
   - Причина (allowed/blocked)

3. **notifications** - Отправленные уведомления
   - Тип сообщения
   - ID сообщения в ВК
   - Статус прочитана

4. **subscription_cache** - Кеш подписок
   - VK User ID
   - Статус подписки
   - Время истечения
